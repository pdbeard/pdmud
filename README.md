# pdmud

TinTin++ MUD client framework with a Zellij layout and custom terminal widgets.

## Requirements

- [TinTin++](https://tintin.mudhalla.net) (`tt++`)
- [Zellij](https://zellij.dev)
- [Go](https://go.dev) 1.21+

## Setup

```bash
git clone https://github.com/pdbeard/pdmud.git
cd pdmud
make build
```

## Run

```bash
make run
```

## Quit

`Ctrl+q` kills the session.
